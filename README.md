# QistonPe – MSME Vendor Payment Tracking Backend

## Tech Stack
- NestJS
- TypeScript
- PostgreSQL / MySQL
- TypeORM / Prisma

## Setup Instructions
- Node.js v18+
- npm install
- Configure `.env`
- Run migrations
- Run seed script
- npm run start:dev
